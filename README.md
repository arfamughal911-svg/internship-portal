# 🎓 Internship Registration Portal

A secure multi-level form validation system built with PHP, MySQL & JavaScript.

## Features
- Client-side validation (JavaScript)
- Server-side validation (PHP)
- SQL Injection prevention (PDO prepared statements)
- XSS prevention (htmlspecialchars)
- AJAX email availability check
- Secure file upload validation
- Password hashing (bcrypt)

## Setup
1. Install WAMP/XAMPP
2. Import `database.sql` into MySQL
3. Place files in `www/internship/`
4. Visit `http://localhost/internship/`

## Tech Stack
- PHP 8+
- MySQL
- JavaScript (Vanilla)
- HTML5 / CSS3
